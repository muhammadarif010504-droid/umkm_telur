# Aplikasi UMKM Telur

Aplikasi desktop berbasis WinForms untuk manajemen stok dan penjualan telur.

## Fitur Utama
- Login pengguna
- Dashboard stok dan penjualan
- Input transaksi toko
- Input stok gudang
- Laporan penjualan

## Teknologi
- Bahasa: C#
- Framework: .NET 10.0
- Database: SQLite (`umkm_telur.db`)

## Cara Menjalankan
1. Buka folder project di Visual Studio
2. Klik **Start** atau tekan `F5`
3. Login menggunakan akun yang tersedia

## Catatan
- Semua data menggunakan satuan **kilo**
- File database tersimpan di folder utama
